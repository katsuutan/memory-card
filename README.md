# Uma Musume Memory Card Game

A memory card game featuring characters from Uma Musume: Pretty Derby. Test your memory by clicking each character only once — the cards shuffle after every click!

## Built With
- React
- Vite
- CSS
- Umapyoi.net API

## Features
- 12 Uma Musume characters fetched from a live API
- Card shuffle animation on every click
- Shake animation on wrong card selection
- Win and lose overlays with Uma Musume themed wording
- Fixed scoreboard tracking current and best score
- Instructions modal
- Responsive layout for desktop, tablet, and mobile
